# bun-react-ssr

To install dependencies:

```bash
bun install
```

To client build:
```bash
bun build ./components/client.tsx --outdir ./public
```

To run:
```bash
bun run server.ts
```

This project was created using `bun init` in bun v1.1.17. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
